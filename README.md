

## Getting Started
First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

این برنامه با استفاده از Next.js و React Query توسعه داده شده و داده‌های بورس اوراق بهادار ایران را برای نمادهای بانک ملت، بانک ملی و پالایش نفت پارس به‌صورت لحظه‌ای نمایش می‌دهد.
داشبورد از طراحی RTL پشتیبانی می‌کند و چهار کارت اصلی دارد که حجم خرید، حجم فروش، قیمت بازار و شاخص‌های نوسان را نشان می‌دهند.

در بخش اصلی، یک جدول وجود دارد که اطلاعات کامل معاملاتی را ارائه می‌کند؛ از جمله اختلاف قیمت خرید و فروش، تغییرات قیمت، تعداد معاملات و حجم‌ها. این داده‌ها هر ۵ ثانیه یک‌بار از طریق API و با مدیریت React Query به‌روزرسانی می‌شوند تا کاربر همیشه آخرین اطلاعات بازار را بدون نیاز به رفرش دستی در اختیار داشته باشد.

این برنامه با تکیه بر اصول UX، از رنگ‌بندی مشخص (سبز برای خرید، قرمز برای فروش و قیمت‌ها، فیروزه‌ای برای نوسانات)، سازماندهی شفاف داده‌ها و چیدمان خوانا استفاده می‌کند تا نمایش داده‌های مالی پیچیده را ساده‌تر کند.
مکانیزم به‌روزرسانی خودکار و طراحی ستونی گسترده باعث می‌شود اطلاعات کلیدی در یک نگاه قابل مشاهده باشد. همچنین رابط واکنش‌گرا و روان، تجربه‌ای مشابه یک داشبورد حرفه‌ای مالی ارائه می‌دهد که برای معامله‌گران و تحلیل‌گرانی که نیاز به مانیتورینگ بلادرنگ دارند، مناسب است
